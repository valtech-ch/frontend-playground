# Blog

## Stack

- React
- Next.js
- Typescript
- Emotion
- Tailwind

## Getting Started

Go into the the project folder:

    cd packages/blog-with-markdown-files

Install the dependencies:

    yarn install

Start the local server:

    yarn dev

Open `http://localhost:3000` in your browser.
